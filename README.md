# Bank Customer Churn Prediction using Artificial Neural Networks (ANN)

## Project Overview

This project predicts whether a bank customer is likely to leave the bank (churn) using an Artificial Neural Network (ANN) built with TensorFlow and Keras.

## Dataset

The dataset contains customer information such as:

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary

Target Variable:

* Exited (0 = Stayed, 1 = Churned)

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Correlation Analysis
4. One-Hot Encoding
5. Feature Scaling using StandardScaler
6. ANN Model Building
7. Model Evaluation

## Technologies Used

* Python
* Pandas

## ANN Architecture

* Input Layer
* Dense Layer (ReLU)
* Dense Layer (ReLU)
* Output Layer (Sigmoid)

## Model Performance

* Accuracy: 86%
* ROC-AUC Score: 0.86
